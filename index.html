UX Writing

In my 5 years working in QA, designs with inefficient or confusing wording had me considering the user well-before entering the UX world. This is how we communicate with our end users, so we must do so effectively.

Accessibility

As a neurodivergent designer, accessibility concerns will always take a front-seat when prioritizing design tasks and issues. When we design universally, we design for everybody, so it should be a top priority.

Web Development

I have experience with HTML, CSS, and a little bit of Javascript. I'm familiar with website platforms such as Wordpress, Wix, and Squarespace, but I prefer to code from scratch, as it allows for the most flexibility.
